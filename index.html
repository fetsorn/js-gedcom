The js-gedcom project is an effort to provide both generic GEDC-format parsing and serialization and FamilySearch GEDCOM 7 specific functionality.

This page has a simple client-side demo FamilySearch GEDCOM 7 validator. Other programmatic features are available on the project page.

To check a GEDCOM file against the FamilySearch GEDCOM 7 specification, pick the file below. The file will not be uploaded to any server: the check happens in your browser on your own computer and will work even if you disconnect from the Internet before you begin.
